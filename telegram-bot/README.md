# NEXTON ADS TV - Telegram Bot

Este es el sistema oficial del bot de Telegram para la plataforma **NEXTON ADS TV**.

## ◈ Características
- Información general de la plataforma.
- Catálogo de series y contenido destacado.
- Enlace de descarga directa de la APK oficial.
- Acceso al canal oficial de novedades.
- Soporte técnico y contacto.

## ◈ Configuración y Ejecución
1. Asegúrate de tener **Node.js** instalado.
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Inicia el bot:
   ```bash
   npm start
   ```

## ◈ Comandos
- `/start`: Mensaje de bienvenida y menú principal.

- `/apk`: Enlace de descarga de la aplicación Android.
- `/canal`: Enlace al canal oficial de Telegram.
- `/soporte`: Información de contacto.
